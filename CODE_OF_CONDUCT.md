# Code of Conduct

## The short version

Be kind. Assume good faith. Critique ideas, not people. If you're unsure
whether something is okay, ask.

## Expected behavior

- Be respectful and welcoming to contributors of all backgrounds and levels.
- Give and accept constructive feedback gracefully.
- Focus on what's best for the project and the community.

## Unacceptable behavior

Harassment, discrimination, personal attacks, or sustained disruption.
Publishing others' private information without consent.

## Scope

Applies in all project spaces (issues, pull requests, discussions) and when
representing the project elsewhere.

## Enforcement

Report concerns privately to **ivan@agorism.org**. Reports are handled
confidentially; the maintainer may warn, remove content, or ban as needed.

This is a lightweight code of conduct in the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/).
